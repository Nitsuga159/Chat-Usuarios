Bienvenido al Chat de Usuarios

WhatsApp 2

Descripción: Este es un chat de usuarios en tiempo real. En cuanto ingrese un nombre de usuario y presione el botón de comenzar podrá charlar con otros usuarios conectados.

Aclaraciones: Mientras haya al menos un usuario conectado, el chat se mantendrá vigente con todos sus mensajes cargados, pero en cuanto se desconecte este último, el chat se reiniciará.
